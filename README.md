**A production website for our family-run blueberry estate nestled in the Carpathian foothills of Ukraine.**

## ✨ Sections

### 🎬 Intro Splash
- Layered logo animation with blur, scale, and draw effects
- Smooth transition into the navbar on completion

### 🎥 Hero
- Dual video background — portrait and landscape variants
- Animated tagline, live season status badge, pulsing CTA

### 🌿 About
- Estate story and mission with highlighted text
- Animated counters and photo lightbox

### 🫐 Varieties
- Three blueberry varieties — Duke, Chandler, Elliott
- Sticky scroll on desktop, sequential reveal on mobile

### 🌱 How We Grow
- SVG-drawn timeline across 5 cultivation steps

### 🖼️ Gallery
- Real field photography with full-screen lightbox viewer

### 📞 Contact
- Phone numbers, Instagram, TikTok, region location

## 🤖 AI-Assisted Development

This project was built using **Claude AI agent in VS Code** as the primary coding assistant.

The workflow looked like this:
- **Me** — design direction, content decisions, UX feedback, testing on real devices
- **Claude** — implementing code based on my requirements and feedback

Every decision was mine. Claude accelerated execution; I owned the vision.

> *"Built with care — for family, with AI."*

## 📚 What I Learned

- Responsive design for real devices — `dvh`, `clamp()`, `env(safe-area-inset-*)`, orientation-aware breakpoints for notches and gesture bars on Android and iOS
- Multi-layer intro splash animation and scroll-triggered transitions with Framer Motion
- Bilingual architecture with `next-intl` — UA/EN with animated language switching
- Custom Cyrillic domain (`ягодакарпат.укр`) registration and Vercel deployment from scratch
